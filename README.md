Flipbook PhoneGap iOS
=====================

Es una versión de slider para para visualizar una revista en iPad

Contenido
_________

* PhoneGap => cordova.ios.js
* jQuery => jquery-1.8.2.min.js
* jQuery Mobile => jquery.mobile-1.2.0.min
* jQuery UI => jquery-ui.js
* Para el manejo de Drag And Drop en iOS => jquery.ui.touch-punch.js
